# BP Panda
A Vue UI Framework Based on iView

## 使用
`npm i bp-panda --save`

在入口文件，index.js
```js
import Vue from 'vue';
import bpPanda from 'bp-panda';
import '../node_modules/bp-panda/styles/index.less';

Vue.use(bpPanda);
```
